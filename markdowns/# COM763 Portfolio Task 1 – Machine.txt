# COM763 Portfolio Task 1 – Machine Learning System Report

## Task Description

Students are required to design, build, evaluate, and demonstrate a complete machine learning system that solves a real-world problem. This task is assessed through a single technical report that documents the full development process from start to finish, along with the working system deployed on Streamlit Community Cloud.

Your report must provide a clear, step-by-step account of how the system was developed, including the reasoning behind every key decision. The work should demonstrate strong engineering practice and experimental rigour, including:

- Problem framing
- Data handling and preprocessing
- Model implementation and debugging
- Performance evaluation and comparison
- Final model selection based on evidence

The report must include visuals throughout (e.g., workflow diagrams, plots, tables) and screenshots of relevant code and outputs to demonstrate implementation progress and technical work.

The aim is for a reader to understand exactly what you did, why you did it, and what you learned from the results.
